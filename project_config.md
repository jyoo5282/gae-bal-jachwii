# 프로젝트 설정

## 프로젝트 정보
- 프로젝트명: 한국 경제 뉴스 수집기
- 버전: 1.0.0
- 생성일: 2024-03-26

## 기술 스택
- Frontend: React.js
- Backend: FastAPI
- Database: SQLite
- 크롤링: Selenium

## 기능 요구사항
1. 주요 경제 뉴스 사이트 자동 수집
2. 실시간 뉴스 모니터링
3. 수집된 뉴스 데이터 저장
4. 웹 인터페이스를 통한 뉴스 표시

## 제약사항
- 뉴스 사이트의 robots.txt 준수
- 과도한 요청 방지를 위한 딜레이 설정
- 중복 뉴스 필터링

## 환경 설정
- Python 3.9+
- Node.js 18+
- 필요한 Python 패키지는 requirements.txt에 명시
- 프론트엔드 패키지는 package.json에 명시
